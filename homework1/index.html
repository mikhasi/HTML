Пользовательские скины в Among Us: Как их получить?

Из этого руководства вы узнаете, как получить пользовательские, или даже создать собственные скины в Among Us совершенно бесплатно.

В Among Us существует небольшое количество стандартных скинов, украшений и питомцев, которые вы можете получить бесплатно. Помимо них, в меню кастомизации своего персонажа перед началом игрового процесса вы можете найти платные украшения для визуализации вашего героя. На данный момент пользователи мобильных устройств могут обойтись лишь существующими скинами, в то время, как игроки на ПК вправе получить классные вещи, просто отредактировав некоторые файлы игры.

Чтобы самостоятельно изменять внешний вид персонажа, вам понадобится скачать программу Assets Bundle Extractor(UABE). Пользовательские скины вы можете найти, перейдя по этой ссылке.

Чтобы установить скины, выполните следующие действия:

  • Откройте Unity Assets Bundle Extractor.
  • Выберите File — Load Package File, затем откройте нужный файл расширения .emip.
  • При запросе базы данных, выберите самый последний пункт.
  • Установите все три флажка (Affected Bundles, Affected Asset Files и Directory).
  • Нажмите Select в нижней части экрана и перейдите в папку Among Us_Data (C\Program Files (x86)\Steam\steamapps\common\Among Us\Among Us_Data).
  • Нажмите «ОК».
  • Выберите File — Save и экспортируйте измененный файл sharedassets0.assets. Вы не сможете заменить существующий файл, пока программа открыта, поэтому сохраните новый файл в удобном для вас месте.
  • Закройте Unity Assets Bundle Extractor, а затем замените стандартный файл sharedassets0.assets на измененный вами в папке Among Us_Data.
  • Запустите игру.
